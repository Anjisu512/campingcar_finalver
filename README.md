# 3조 캠핑카 프로젝트 깃허브
- Push전엔 뭐다? Upstream Pull, Cach Clear를 한다!
- 스프링 프로젝트 같은 경우 각종 잡다한 파일이 딸려 올라올 수 있으므로 Push전에 터미널로 캐시 삭제 할 것.
- 이와 관련해 이그노어를 중간에 하게되면 캐시가 꼬일 수 있기때문에 수정 후엔 따로 알리겠음.
- 언제 터질지 모르기때문에 백업을 생활화하자!
